# Scientific Calculator (Python Tkinter)

A graphical scientific calculator built using Python and Tkinter.  
This calculator provides basic arithmetic operations along with advanced mathematical functions such as logarithms, trigonometric functions, square roots, and powers.

## Features

- Addition, subtraction, multiplication, division
- Square root calculation
- Power (x^y)
- Logarithmic calculations (log10)
- Trigonometric functions (sin, cos, tan)
- Dark theme graphical interface
- Clickable buttons for all operations
- Error handling for invalid inputs

## Technologies Used

- Python 3
- Tkinter (GUI library)
- Math module

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/scientific-calculator

2. Navigate to the folder

cd scientific-calculator

3. Run the program

python3 scientific_calculator.py
